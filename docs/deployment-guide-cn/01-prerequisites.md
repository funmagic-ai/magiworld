## 前置要求

- 具有管理员权限的 AWS 账户
- 已安装并配置 AWS CLI
- 已添加域名的 Cloudflare 账户
- 在 AWS EC2 控制台创建的 SSH 密钥对
- 您的域名（例如 `funmagic.ai`）

> **重要提示：永远不要在应用程序中使用 Root 用户**
>
> 在应用程序中使用 AWS root 用户凭证是严重的安全风险：
> - Root 用户对您的整个 AWS 账户拥有**无限制访问权限**
> - 凭证泄露可能导致**整个账户被入侵**
> - Root 用户无法通过 IAM 策略进行限制
>
> 始终为每个应用程序创建具有最小权限的专用 IAM 用户。
